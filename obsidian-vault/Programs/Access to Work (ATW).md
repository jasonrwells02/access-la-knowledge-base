# Access to Work (ATW)

Standing order service for customers commuting to employment or job training sites. Trips are scheduled based on work start times so customers arrive on time.

## Eligibility
Must use the program exclusively for travel to/from employment or job training. Must maintain a consistent weekly schedule.

## Benefits
- Trips scheduled to work start times for on-time arrival
- Flexible modifications if work hours change (contact Standing Order department, no reapplication needed)
- Standing Order service (no daily calling required)









**Fare:** Standard Access fare
